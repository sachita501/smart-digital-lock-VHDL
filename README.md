# smart-digital-lock-VHDL
Smart Digital Lock System using VHDL with password verification, 7-segment display, alarm, and master reset. Simulated using Vivado software.
# Smart Digital Lock System Using VHDL

## Objective
Design and simulate a smart digital lock system using VHDL.

## Features
- Password verification
- Simulated keypad input
- 7-segment display output
- Wrong attempt detection
- Alarm after three incorrect attempts
- Master reset

## Tools Used
- Vivado 2026.1
- VHDL

## Files
- smart_lock.vhd
- seven_segment.vhd
- smart_lock_tb.vhd

## Simulation
The design was verified using Vivado Behavioral Simulation. No FPGA hardware was used.
